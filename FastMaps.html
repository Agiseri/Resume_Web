FastMaps(Senior Design Project): Mass transit app for the DFW Metroplex

Source code

  • Converted static GTFS data and into a GeoJSON format to create a custom map illustrating public transportation networks using Mapbox Studio.
  • Combined Mapbox APIs to enable turn by turn navigation instructions as well as predictive address searching.
  • Applied Dijsktra algorithm in python to GTFS data to find optimal routes between bus/train stops
